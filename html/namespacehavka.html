◆ ResponseType

enum havka::ResponseType

Enum for response type.

Enumerator
PostSuccess 

Post request was successfully processed.

ErrorWhilePosting 

Being sent only if posting was safe (RequestType::PostMessageSafe) and error occured

GetSuccess 

Get request was successfully processed.

EmptyTopic 

If get-request was blocking, connection is alive and waiting for a message. If get-request was non-blocking, std::nullopt is returned

Error 

Unknown error.
